## 📌 Project Overview

The **Hotel Booking Form** is a responsive and user-friendly web application built with **HTML5, CSS3, and Vanilla JavaScript**. It allows users to book hotel rooms online by selecting check-in/check-out dates, choosing room types, setting the number of guests, and submitting special requests — with real-time price calculation and a booking confirmation popup.

## 💡 How It Works

```
1. 🛏  User selects a Room Type (e.g., Deluxe Suite — $280/night)
        ↓
2. 📅  User picks Check-In and Check-Out dates
        ↓
3. 🔢  System calculates number of nights automatically
        ↓
4. 💰  Total price updates live (Room Rate × Nights + 12% Tax)
        ↓
5. 👤  User fills in personal details (name, email, phone)
        ↓
6. ✅  Form validates all fields with inline error messages
        ↓
7. 🎉  Confirmation popup shows unique Booking Reference ID
```
